# Image Assets Required

Place the following images in the `public/images/` directory:

## Homepage Images
- `hero-headphones.jpg` - Hero section background image with XX99 Mark II headphones
- `category-headphones.png` - Headphones category thumbnail (transparent PNG)
- `category-speakers.png` - Speakers category thumbnail (transparent PNG)
- `category-earphones.png` - Earphones category thumbnail (transparent PNG)
- `product-zx9-speaker.png` - ZX9 speaker product image (transparent PNG)
- `product-zx7-speaker.jpg` - ZX7 speaker background image
- `product-yx1-earphones.jpg` - YX1 earphones image
- `about-image.jpg` - Person enjoying audio (about section)

## Temporary Placeholders
Until you add actual images from your Figma design, you can:
1. Export images from Figma
2. Use placeholder services temporarily
3. Continue with development (images will show as broken until added)

## Image Specifications
- Hero: 1440x729px (desktop), responsive
- Category thumbnails: 300x300px (transparent)
- Product images: Various sizes based on Figma design
- All images should be optimized for web (use WebP where possible)
